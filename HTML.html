超文本标记语言（HTML）

什么是HTML？

HTML ，即HyperText Markup Language（超文本标记语言）。我们平时所说的网页，即为使用HTML语言来书写的文件，也叫做超文本文档。

如何创建HTML？

HTML的本质是一个文本文件，我们只需要建立一个以html或htm结尾的文件，然后在里面书写一些内容，就能通过浏览器来展示这些内容，这就已经创建了最简单的HTML。

HTML标签

HTML是超文本标记语言，可以用标记来实现一些普通文本实现不了的操作（超文本操作），这里所说的标记，就是指标签。

HTML标签，一般由开始标签、标签体、结束标签和标签属性组成。

关于标签的一点说明：
部分标签没有标签体和结束标签，HTML属性都有其默认值，因此也不是必须的部分。另外，由于HTML标签的不规范性，有些标签可以有多种书写方式。按照规范要求，HTML标签采用小写字母书写。HTML的属性部分可以使用双引号，也可以使用单引号。

一个最基本的HTML结构如下：

<!doctype html>

<html>

<head>

    <title>网页标题</title>

</head>

<body>

    网页主体（网页正文）

</body>

</html>

HTML的运行过程

文档类型声明

HTML语言的发展经历了一个很长的过程，为了帮助浏览器正确的解析HTML文件，每个HTML文件都需要声明自己的文档类型。1991年开始出现Html，接着2.0，继续3.0,1994年升级为4.0版本，直到现在的html5.

HTML5文档类型（目前最常用的文档类型）

<!DOCTYPE html>    HTML也有很多个不同的版本，只有完全明白页面中的使用的确切HTML版本，浏览器才能完全正确地显示出HTML页面

HTML4.01严格类型（不支持过期标签和框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML4.01过渡类型（支持过期标签，不支持框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

HTML4.01框架集类型（支持过期标签和框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

网页头部

网页头部指的是<head></head>中的部分，这一部分会首先被浏览器加载，一般用于书写一些设置性的内容。

<meta>标签

一般需要写在<head></head>中，是网页的元信息。一般有两个属性：name和http-equiv.name属性用于描述网页信息，对应有content属性。http-equiv多用于规定页面内容使用的文字语言

<meta name="Author" content="claireHou">告知搜索引擎当前站点Chrome.sublime-build的作者

指定网页为UTF-8编码（兼容性较好）html4.0版本写法：

<meta http-equiv="content-type" content="text/html;charset=utf-8">

指定网页为UTF-8编码（HTML5专属）：

<meta charset="utf-8">

<title></title>标签

当前网页的标题，会出现在浏览器的标题栏/标签栏。也会作为将当前网页添加到收藏夹/书签时使用的默认名称。

按照HTML规范，<title></title>是唯一需要包含到<head></head>中的标签。

网页主体

网页主体指的是<body></body>中的部分，一般用于书写一些需要给用户展示的内容。一般由标签符号构成。 常用标签：h1-h6,p,br,img,span,div,a,三种列表标签，table，form以及特殊字符

字符实体

向下面这样，给书写在body的内容添加一些空格。

<body>

    这 是一些内     容。

</body>

使用浏览器打开这个文件，可以发现：“这”和“是”中间的单个空格是正常显示的，而“内”和“容”之间的多个空格会显示成只有一个空格。这是由于浏览器会自动截短大于一个的空格，目的是为了让程序员更精确的控制文本格式。

如果想在HTML中使用不会被截短的空格，要使用字符实体&nbsp;表示。

字符实体 显示结果 描述
&nbsp;   空格
&lt; < 小于号
&gt; > 大于号
&amp; & 和号
&yen; ¥ 元
&copy; © 版权（copyright）

<br>和<hr>

像多个空格一样，直接出现在HTML中的回车换行也会被截短成一个空格。如果想在HTML中使用不会被截短的换行，要使用标签<br>代替。

如果添加一个占据一行的水平线，可以使用<hr>。

图片（重点）

在网页中，可以使用<img>标签来显示一个图像。
对于一个图像，常见的属性有三个：

src

图像的来源，可以是一个地址，也可以是一段base64码。

<!--可以是一个网络地址-->

<img src="https://www.baidu.com/img/bd_logo1.png?qua=high">



<!--可以是一个本地地址（一般采用相对路径）-->

<img src="../image/logo.png">



<!--可以是一段base64码（base64码过于庞大，这里仅为示例，无法正常显示）-->

<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAhwAAAEC...">

在线将图片转换为base64码

alt

图像的代替文字，当图像无法正确加载时，用于提示用户图片内容的文字。

<img src="../image/logo1.png" alt="百度一下，你就知道！">

title

不只是图像，所有的标签都可以添加这个属性用于增加一个悬浮提示。

<img src="../image/logo.png" title="百度一下，你就知道！">

<img src="../image/flower.jpg" width="200px" height="200px">

target属性

指定页面在何处打开（不要漏写下划线）：
_blank：在新窗口打开。
_self：在当前窗口打开。

跳转到网页某一部分

可以使用链接直接跳转到网页某一部分，这叫做锚点跳转。

声明锚点位置的方式：

  1. 任何一个标签的id属性（注意：id属性标签唯一）。
  2. 页面顶部（无需声明，固定为#top）。HTML5新增内容模块

<!--跳转到其他页面的锚点-->

<!--跳转到页面顶部，无需声明锚点-->

<a href="#top">回到顶部</a>

表单（特别重点）

表单用来提供一些可以由用户选择或输入的元素，主要用于收集用户数据提交给服务器，比如登陆注册。

表单由一个form标签和若干个其他表单元素标签组成（其他表单元素可以不依赖form单独使用）。用户在填写其他表单元素后，可以提交整个表单中name属性不为空的表单元素中的数据到服务器。

<form></form>标签

action属性规定表单提交的地址。
method属性规定表单提交的方式（取值可以为get或post，之后详细说明）。
enctype属性指定数据的编码处理方式（当method=”post”时适用）。

<input>元素（重要）

最常见的表单元素，根据其type属性的不同取值，可以显示为不同的输入组件。 text文本框，password密码框，radio单选框，checkbox复选框，file文件域，textarea文本域，submit提交按钮，reset重置按钮，button普通按钮。特别注意：当这个按钮处在一个表单内部时，如果不手动指定button的type属性，浏览器可能会将其当做普通按钮或者提交按钮。因此，这时必须为这个标签指定type属性。H5新增一些：placeholder，color，range，number,email,autofocus，tel等属性用于在用户输入前提示用户。

标签元素分类：块级元素、行内元素（重点）

块级元素 display属性:block

  1. 独占一行，默认情况下，其宽度自动填满其父元素宽度
  2. 可以设置宽和高，设置了宽度还是独占一行
  3. 可以嵌套块级元素及行内元素。可以设置margin和padding属性
  4. 可以设置margin和padding属性
  5. 可以互相转换display:inline变成行内元素

浏览器默认行为下常见的块级元素：div、h1~h6、hr、p、table、form。

行内元素 display属性:inline

  1. 不会独占一行，相邻行内元素会排列在同一行里，直到一行排不下，才会换行，其宽度随元素的内容而变化
  2. 不可以设置宽和高
  3. 水平方向的内外都产生边距效果但竖直方向的内外边距却不会产生边距效果
  4. 在可以有标签体时，内部只能有其他行内元素或文字，不可以嵌套块级元素
  5. 可以互相转换display:block变成行块级元素

浏览器默认行为下常见的行内元素：span、img、font、a、b、i、sub、sup、input、button。

内联块元素 display属性:inline-block

浏览器默认行为下常见的内联块元素：img、input、select

标签嵌套规则：

  1. 不能交叉嵌套
  2. 块级元素嵌套行内元素或者块级元素
  3. 行内元素不能嵌套块级元素
  4. 块级元素不能放在<p>里面
  5. 有几个特殊的块级元素只能包含行内元素，不能再包含块级元素，这几个特殊的标签是： h1、h2、h3、h4、h5、h6、p、dt
